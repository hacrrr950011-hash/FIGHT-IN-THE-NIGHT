# ⚔️ Fight in the Night - PvP Fighting Game

A cutting-edge mobile fighting game with PvP battles, AI opponents, and rich progression systems.

## 🎮 Features

### Game Modes
- **Jannu Mode**: Battle against AI opponents
- **Pro Mode**: Real-time PvP battles

### Maps
- 🌴 Jungle
- 🏙️ City
- 🏜️ Desert
- 🚀 Space
- ❄️ Snow

### Progression
- 100-level system
- Character unlocks
- Pet system
- Shop with XP purchases
- Anti-cheat system

### Multi-Language Support (5 Languages)
- 🇬🇧 English
- 🇮🇳 हिंदी (Hindi)
- 🇮🇳 తెలుగు (Telugu)
- 🇮🇳 ગુજરાતી (Gujarati)
