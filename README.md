## 📱 Installation

```bash
git clone https://github.com/hacrrr950011-hash/FIGHT-IN-THE-NIGHT.git
cd FIGHT-IN-THE-NIGHT
npm install
```

### Build APK

**Option 1: Local Build (Recommended)**
```bash
npm run build-android
```

**Option 2: Online (AppsRunnerIO)**
1. Visit https://appsrunner.io
2. Paste repository URL
3. Build APK
4. Download & Install

**Option 3: Expo Go (Testing)**
```bash
npm start
```
Then scan QR code in Expo Go app
